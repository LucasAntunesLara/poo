# Programação Orientada a Objetos

## Como instalar o Typescript

Caso queiro instalá-lo apenas localmente:

```sh
npm install typescript --save-dev
```

Senão, é possível realizar a instalação de forma global:

```sh
npm install typescript
```

## Como transpilar o código Typescript para Javascript

```sh
npx tsc nomeDoSeuArquivo.ts
```

_OBS_: se você tem o Typescript instalado globalmente na sua máquina, não é necessário usar o npx.

## Como rodar o arquivo Javascript

1. Tenho o Node instalado

2. Rode o comando:

```sh
    node nomeDoSeuArquivo.ts
```
